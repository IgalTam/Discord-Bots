Bad Grouper Scratchwork File

mult design:

	available commands: 
	- make x name='BG_Group_' role_make=True
	- clean name='BG_Group_'
	- mtr role [usr1, usr2, ...]
	- rmr role [usr1, usr2, ...]
	- make_role role_name
	
	mult command line args (follow getopt format):
	
		flags:
		-h: help, overrides other options
		-r [name='BG_Group_']: create role with name
		-m [name='BG_Group_']: create tcs & vcs with name 
		-c [name='BG_Group_']: clean all channels & roles with name
		-v [name]: move all listed users with role, list at end of command
		-g: give role to users, requires -v
		-e: remove role from users, requires -v
